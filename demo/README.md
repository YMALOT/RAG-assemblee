# RAG sur les débats parlementaires — démo

Démo interactive d'un système RAG construit sur les débats de l'Assemblée
nationale concernant la loi contre le harcèlement scolaire (loi n°2022-299).

Tapez une question, le système récupère les passages pertinents dans les
débats et génère une réponse en citant ses sources. Un panneau « coulisses »
affiche les chunks récupérés.

**👉 Code complet, méthodologie d'évaluation et résultats chiffrés :**
[https://github.com/YMALOT/RAG-assemblee](https://github.com/YMALOT/RAG-assemblee)
